# iiCourse 正式发布！

经过数月的开发和测试，**iiCourse** 终于正式发布啦！

## 这是什么？

iiCourse 是一款专为智慧山商设计的桌面应用，让你告别繁琐的网页操作，一键查询课表、成绩、空教室等信息。

## 主要功能

- **📅 课程表查询** - 查看本周课表，支持多周切换
- **📊 成绩查询** - 快速查询各科成绩，自动计算 GPA
- **🏫 空教室查询** - 查找空闲教室，自习好去处
- **💳 一卡通余额** - 实时查询余额和消费记录
- **⭐ 一键评教** - 快速完成学生评教
- **🔐 CAS 登录** - 支持智慧山商统一认证

## 技术亮点

- 基于 .NET 9 和 WPF 开发
- 使用 HtmlAgilityPack 解析 HTML
- DES 加密处理登录认证
- 异步操作，界面流畅不卡顿
